# class_library
tp5快速开发常用类库
#### 修改composer.json
```
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/skj198568/class_library.git"
        }
    ]
```
#### 执行命令
tp5.0
```
    composer require "skj198568/class_library:1.0.*"
```
tp5.1
```
    composer require "skj198568/class_library:1.1.*"
```
